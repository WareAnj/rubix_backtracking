# Rubix Backtracking
A Backtracking Project in completion for CMSC 142 in University of the Philippines Los Banos

## Requirements
- GCC 5 or better

## How to Run

```bash
npm start
```

## Rotation Guides

### Movement
```js
0 :: UP
1 :: LEFT
2 :: TOP
3 :: RIGHT
4 :: BOTTOM
5 :: DOWN
```

### Colors
```js
white  :: ascii 87 :: 0
red    :: ascii 82 :: 1
blue   :: ascii 66 :: 2
orange :: ascii 79 :: 3
green  :: ascii 71 :: 4
yellow :: ascii 89 :: 5
```
